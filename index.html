명령어

명령어 형식

  1. mute
    대상을 채팅금지시킵니다.
    !mute @대상 이유
  2. tempmute
    대상을 일정 기간동안 채팅금지시킵니다. 시간형식 지정법은 여기를 참고하세요.
    !tempmute @대상 기간 이유
  3. clear
    채팅을 특정 개수만큼 지웁니다
    !clear 개수
    혹은 특정 사용자의 채팅을 최근 채팅부터 특정 개수만큼 지웁니다.
    주의) 해당 사용자가 서버를 나갈 경우 오류로 인해 다른사람이 친 채팅을 포함해서 해당 개수만큼 지워집니다. 조심해서 사용하세요
    !clear @대상 개수
  4. infractions
    대상의 경고 기록을 확인합니다.
    !infractions @대상
  5. warn
    대상을 경고합니다
    !warn @대상 이유
  6. slowmode
    채널의 채팅 슬로우모드를 수정합니다. 시간형식 지정법은 여기를 참고하세요
    !slowmode 시간

시간 형식

단위 문자
초 s
분 m
시 h
일 d
주 w

예시) 1초 : 1s 3분 : 3m 1분 30초 : 90s

주의사항

  1. 장난으로 진짜 뮤트나 경고 금지
  2. 슬로우모드 없어도 단타 금지
  3. 규칙은 최대한 암기
  4. #관리사무소 이외의 곳에서 !infractions쓰지 않기
  5. 경고 기준에 애매한 것 같으면 꼭 #관리사무소에서 논의하기
  6. 서버장 호출에 부담갖지 않기
